# 📚 StudyDash

A modern, responsive study dashboard built with **React**, **TypeScript**, **Vite**, and **TailwindCSS**.  
StudyDash helps learners organize their study materials, track progress, and stay productive with a clean and intuitive interface.

---

## 🚀 Features

- ⚡ Fast development with Vite  
- 🎨 Customizable UI using TailwindCSS  
- 🛠️ TypeScript support for type safety  
- 📂 Organized project structure (`src`, `public`, configs)  
- 🔄 Hot Module Replacement for rapid iteration  
- 🌐 Responsive design for desktop and mobile  

---

## 📦 Tech Stack

| Technology      | Purpose                          |
|-----------------|----------------------------------|
| **React**       | UI framework                     |
| **TypeScript**  | Strong typing & maintainability  |
| **Vite**        | Fast bundler & dev server        |
| **TailwindCSS** | Styling & responsive design      |
| **PostCSS**     | CSS transformations              |

---

## 🛠️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/nupoor-mahajan/studydash.git
cd studydash
npm install
```
---

## ▶️ Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```

## 📂 Project Structure

```text
studydash/
├── public/               # Static assets
├── src/                  # Source code
│   ├── components/       # Reusable UI components
│   ├── pages/            # Page-level components
│   ├── styles/           # Global styles
│   └── main.tsx          # Entry point
├── index.html            # Root HTML file
├── package.json          # Dependencies & scripts
├── tailwind.config.js    # Tailwind configuration
├── vite.config.ts        # Vite configuration
└── tsconfig.json         # TypeScript configuration
```
